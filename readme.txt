You can change date of post using this plugin

== Description ==

This plugin add metabox to a post edit page. There are you can change date of a post. The plugin works with Genesis Framework only.

== Installation ==

1. Upload the `change-date` folder to the `/wp-content/plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Why do I need this plugin? =

The plugin can to change a post's date works with Genesis Framework only.
